# Instagram-Clone

以 Firebase 為資料庫模仿 Instagram app，實現註冊登入、檢視文章、發佈貼文、留言按讚、追蹤他人、通知訊息及儲存貼文...等功能，藉由 Singleton 來管理 Firebase 套件，並嘗試 MVVM 的 data binding 來綁定 data 與 View。
